# 欢迎来到“语言拼图”游戏
在**语言拼图**中，你可以充分发挥你的常识和语言能力，造出非常有水平的句子，用有限的回合内达到更高的得分，看谁可以成为语言大师。
- **版本**: 0.1 beta
- **作者**: [Bill Han]
希望你能够在这个游戏中找到更多灵感。
请告诉我您希望单词池中的单词范围，以及语种。比如：“地名，中文”

**支持各种语言，包括中文/英文/日语/韩语……**

**特别适合给小朋友学习外语用。**

推荐模型：GPT4.0

### 点击[链接](sentence-making.txt)，将所有代码拷贝到GPT4对话框中，即可开始体验游戏
